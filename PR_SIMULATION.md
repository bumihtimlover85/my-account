# PR 流程模拟

> **关联 Issue**: #41
> **说明**: 这只是个测试，请审查者和我一起模拟一下 PR 流程里的各项功能，来回来两三个回合。

## 用途

此 PR 仅用于模拟 GitHub PR 流程，包括但不限于：

- [x] 代码审查 (Code Review)
- [ ] 评论/回复 (Comments & Replies)
- [ ] 请求变更 (Request Changes)
- [ ] 批准 (Approve)
- [ ] 合并 (Merge)

## 指南

审查者可以在代码行上添加评论、提出问题、或是批准此 PR。我会相应回应，共同完成一次完整的 PR 流程模拟。
